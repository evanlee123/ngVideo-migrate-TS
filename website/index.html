ngVideo Angular video player Web Component — powered by Angular Elements

Quick Start

Build the Web Components bundle, then open this page to see the demo:

npm install
npm run build:elements
# open website/index.html in a browser

Usage

Include the bundle script and use the custom elements in any HTML page:

<script src="ng-video-bundle.js"></script>

<ng-video class="video" id="player">
  <vi-controls class="controls"></vi-controls>
  <vi-feedback class="feedback"></vi-feedback>
  <vi-playlist class="playlist"></vi-playlist>
</ng-video>

<script>
  customElements.whenDefined('ng-video').then(() => {
    const player = document.getElementById('player');
    player.addSource('mp4', 'https://example.com/video.mp4');
  });
</script>

Configuration

Set options before loading the script:

<script>
  window.ngVideoConfig = { REFRESH: 100, VOLUME_STEPS: 0.2 };
</script>
<script src="ng-video-bundle.js"></script>

Components

<ng-video>        — Container: hosts <video> element, manages playback
<vi-controls>     — Play/Pause buttons
<vi-feedback>     — Status overlay (time, buffering, loading)
  <vi-volume>     — Volume bar with +/- controls
  <vi-timeline>   — Seekable range slider
  <vi-buffer>     — Canvas-drawn buffer visualization
  <vi-messages>   — Event message counter
<vi-playlist>     — Video list with selection
  <vi-meta>       — Per-video duration metadata
